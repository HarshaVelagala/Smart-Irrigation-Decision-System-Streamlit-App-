# 🌿 Smart Irrigation Decision System
### Streamlit Web Application

---

## Quick Start

```bash
# 1. Install dependencies
pip install streamlit pandas

# 2. Run the app
streamlit run app.py

# App opens at → http://localhost:8501
```

---

## What the App Does

| Step | Action |
|------|--------|
| **Upload** | Accept `soil_sensor_data.csv` + `weather_data.csv` |
| **Validate** | Check all required columns exist |
| **INNER JOIN** | Merge on `[date, field_zone]` — only fully matched records proceed |
| **Feature Engineering** | Compute 5 agronomic features |
| **Recommend** | Rule-based engine → 4 irrigation levels |
| **Display** | KPI cards, recommendation table with explanations, zone summary, water bars |
| **Export** | Download full results + zone summary as CSV |

---

## Required CSV Schemas

**soil_sensor_data.csv**
```
date, field_zone, soil_moisture_pct, soil_temp_celsius, crop_type,
soil_ph, nitrogen_ppm, phosphorus_ppm, potassium_ppm
```

**weather_data.csv**
```
date, field_zone, temperature_max_c, temperature_min_c, humidity_pct,
rainfall_mm, evapotranspiration_mm, wind_speed_kmh, solar_radiation_mj,
weather_condition
```

> Use the CSV files generated by `src/generate_datasets.py` from the companion pipeline project.

## Recommendation Logic

| Level | Volume | Trigger |
|-------|--------|---------|
| 🔴 Irrigate Now | 25 mm | Deficit > 20% OR negative balance + 5 dry days |
| 🟠 Irrigate Soon | 15 mm | Deficit 10–20% OR 3+ dry days |
| 🟡 Light Irrigation | 8 mm | Deficit 1–10% OR any dry day |
| 🟢 No Irrigation | 0 mm | Adequate moisture and positive balance |
